# 国际化 (i18n)

## 功能 ID

`FEAT-I18N-001`

## 功能描述

Mobazha 国际化系统支持 9 种语言，为全球用户提供本地化体验。

## 支持语言

| 代码 | 语言       | 本地名称  |
| ---- | ---------- | --------- |
| `en` | English    | English   |
| `zh` | Chinese    | 中文      |
| `ja` | Japanese   | 日本語    |
| `ko` | Korean     | 한국어    |
| `es` | Spanish    | Español   |
| `fr` | French     | Français  |
| `de` | German     | Deutsch   |
| `ru` | Russian    | Русский   |
| `pt` | Portuguese | Português |

### 使用方法

#### 1. 在组件中使用 Hook

```tsx
import { useI18n } from '@mobazha/core';

function MyComponent() {
  const { t, locale, setLocale, formatNumber, formatCurrency, formatDate } = useI18n();

  return (
    <div>
      <h1>{t('common.welcome')}</h1>
      <p>{t('products.count', { count: 10 })}</p>
      <p>{formatCurrency(99.99, 'USD')}</p>
      <p>{formatDate(new Date())}</p>

      <select value={locale} onChange={e => setLocale(e.target.value)}>
        <option value="en">English</option>
        <option value="zh">中文</option>
      </select>
    </div>
  );
}
```

### API 参考

#### useI18n Hook

```typescript
interface UseI18nReturn {
  // 当前语言
  locale: Locale;

  // 设置语言
  setLocale: (locale: Locale) => void;

  // 翻译函数
  t: (key: string, params?: Record<string, string | number>) => string;

  // 数字格式化
  formatNumber: (num: number, options?: Intl.NumberFormatOptions) => string;

  // 货币格式化
  formatCurrency: (amount: number, currency: string) => string;

  // 日期格式化
  formatDate: (date: Date, options?: Intl.DateTimeFormatOptions) => string;

  // 相对时间格式化
  formatRelativeTime: (date: Date) => string;
}
```

## 添加新语言

1. 在 `packages/core/i18n/types.ts` 中添加语言代码到 `SUPPORTED_LOCALES`
2. 在 `packages/core/i18n/locales/` 中创建新的语言文件
3. 在 `packages/core/i18n/locales/index.ts` 中导出新语言

示例：

```typescript
// types.ts
export const SUPPORTED_LOCALES = [
  'en',
  'zh',
  'ja',
  'ko',
  'es',
  'fr',
  'de',
  'ru',
  'pt',
  'it',
] as const;

export const LOCALE_INFO: Record<Locale, { name: string; nativeName: string }> = {
  // ...
  it: { name: 'Italian', nativeName: 'Italiano' },
};

// locales/it.ts
export const it: TranslationResource = {
  common: {
    welcome: 'Benvenuto',
    // ...
  },
  // ...
};
```
